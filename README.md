# ⚔️ 數學闖關對戰 🦊 VS 🐼

一款給小學生玩的雙人數學對戰遊戲，適合在**平板橫放**時兩人各坐一側同時遊玩。

## 玩法

- 兩位玩家各自在平板的一側作答，**不用等對方**，可以一直往下算。
- 答對 **+10** 能量，答錯 **-5** 能量。
- 一局 **15 分鐘**，時間到比能量值高低決定勝負。
- 答題後會出現開心 🥳 或哭哭 😢 的角色與彩帶動畫，中間的賽道上 🦊 和 🐼 會依分數互相追趕。

## 題庫

共 88 題，涵蓋：

- 正方體、長方體的表面積（含立體圖、展開圖、堆疊體、切割體等課本圖形）
- 長度、重量、面積單位換算（公里／公噸／公頃／公畝等）

立體圖形直接取自題本，確保尺寸數值完整呈現。

## 怎麼玩

直接用瀏覽器開啟 `index.html` 即可，無需安裝。也可透過下方的線上網址直接遊玩。
